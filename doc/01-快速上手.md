# 1. 快速上手

## 界面布局

- **底部居中工具条**：所有元件工具（图标，鼠标悬停看名字和快捷键）。
- **顶栏卡片**：
  - 左：`ChipSim` 品牌 · `模板▾` 下拉（选一个结构盖章使用，🗑 删自定义模板）。
  - 右：`↶ ↷` 撤销/重做 · `🔗 分享`（生成链接并复制）· `文件▾`（保存/读取/导出/导入/清空）· `🌙/☀️` 深浅主题 · `?` 帮助。
- **右下角**：模拟**速度**滑条（每帧跑多少 tick）。
- **画布**：无限平移缩放，背景是会随缩放细分的点阵。

## 鼠标 / 键盘

| 操作 | 作用 |
| --- | --- |
| 左键点 / 拖 | 用当前工具放置（画线工具可拖动连续画） |
| 滚轮 / 触摸板捏合 | 缩放 |
| 空格拖动 / 中键拖动 / 双指滑动 | 平移 |
| 右键 | 上下文菜单（对选中元件：复制/剪切/副本/粘贴/旋转/删除、真值表、存为模板） |

**工具快捷键**（也对应底部图标）：

`V` 选择/移动 · `H` 操作/探针 · `W` 线 · `A` 与 · `O` 或 · `X` 异或 · `N` 非 ·
`B` 按钮 · `L` 灯 · `K` 时钟 · `D` 触发器 · `G` 桥 · `U` 总线 · `M` 合并 · `S` 拆分 · `Y` 显示 · `E` 删除。

**编辑快捷键**：`R` 旋转 · `Ctrl+Z/Y` 撤销/重做 · `Ctrl+C/X/V` 复制/剪切/粘贴 · `Ctrl+D` 复制副本 · `Del` 删除选中。

> 两种"鼠标态"要分清：**V（选择）**用来选/移动/框选元件；**H（操作）**用来点按钮、悬停看值、把线钉到波形图。建电路时大多在放置工具和 V 之间切，玩电路时切到 H。

## 第一个电路：与门点灯

目标：两个按钮都按下时，灯才亮（A AND B）。

最快的方式是**直接盖一个内置模板再补上输入输出**，但这里先手动走一遍，理解"门怎么接"。

1. **放两个按钮**：按 `B`，在画布左侧点两下，竖着放两个按钮（隔开一格，别贴在一起——贴一起会连成同一个网）。
2. **放一个与门**：按 `A`。注意**与/或/异或门是 2 格高的**：左上、左下各一个输入，右侧一个输出。把它放在两个按钮右边。
3. **连线**：按 `W`（线工具）。
   - 在色板里选 **红**，从上面那个按钮拉一条红线到门的**左上**输入。
   - 在色板里选 **蓝**，从下面那个按钮拉一条蓝线到门的**左下**输入。
   - ⚠️ 两个输入挨着，必须用**不同颜色**，否则两根线会短在一起（见 [连线·颜色·总线](04-连线-颜色-总线.md)）。
4. **放灯**：按 `L`，放在门的右侧输出处；用线把门的输出连到灯。
5. **跑起来**：按 `H` 进操作模式，依次点两个按钮。两个都亮（=1）时，灯才亮。

> 看不清哪根线通没通？在 **H** 模式把鼠标停在任意线/灯上：会**点亮整张相连的网**并显示当前值。

## 更快：盖模板

顶栏 `模板▾` 里有现成的 **NAND / NOR / XNOR / 半加器 / T 触发器**。选一个，鼠标会跟着一个半透明预览，点击即"盖章"放下（`R` 可旋转）。盖下来的是**真实可见、可改**的电路——双击不了但你可以选中、拆开看它内部怎么连，这是学习的最佳方式。

接下来建议读 [工作原理](02-工作原理.md)，理解模拟是怎么跑的——这能帮你避开绝大多数"为什么我的电路不对"的坑。
