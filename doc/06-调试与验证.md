# 6. 调试与验证

三件法宝：**探针**（看某根线）、**时序波形**（看随时间变化）、**真值表**（验组合逻辑）。都在 **操作模式 `H`** 下使用。

## 探针：悬停看值 + 点亮整张网

切到 **H**，把鼠标**停在**任意线/引脚/元件上：

- **点亮整张相连的网**（高亮描边）——一眼看清"这些线其实是同一根"，排查"以为接上了其实没接 / 以为没接其实短了"。
- 旁边显示该网**当前的值**（总线额外显示 16 进制）。

这是最快的排错手段：结果不对时，先探一探关键线，确认连接和值符合预期。

## 时序波形（逻辑分析仪）

理解时钟、触发器、计数器的**时序**全靠它。

- 在 **H** 模式下**点击一根线**，把它**钉选**到底部的滚动波形面板；再点一次取消。
- 想钉**按钮/时钟**本身：**Shift + 点击**它们。
- 最多 8 路。1 位信号画成方波；总线画成多级阶梯并标 16 进制值。
- 横轴是 tick（每个 tick 采一次样），最新在右侧滚动。

典型用法：把 **时钟** 和 **触发器的 Q** 一起钉上，就能直观看到"Q 在每个时钟上升沿翻转"、计数器逐位二分频等。**信号流动动画**（通电线上的走马灯）也帮你看出哪条线带电、电流往哪走。

> 看不清传播过程时，把右下角**速度调慢**，信号一级级走、波形一格格推，看得很清楚。

## 自动真值表

验证一个**组合**电路块对不对：

1. 用 **V** 框选这个块——块里要包含 **按钮（当输入）** 和 **灯（当输出）**。
2. **右键 → 真值表**。
3. 弹窗会**枚举所有输入组合**，自动跑模拟，列出每种输入下的输出（输入按位置记作 A、B…，输出记作 O1、O2…）。`Esc` 或点外面关闭。

把你搭的半加器/全加器框上按钮和灯，一键核对真值表，比手点快得多。

> 真值表只对**组合逻辑**有意义（输出只由当前输入决定）。如果块里有反馈/触发器（带记忆），真值表不代表完整行为，那种情况用**时序波形**看。

## 保存、分享、找回

- **自动存**：改动会自动存到浏览器 localStorage，刷新还在。
- **文件▾**：`保存/读取` 走本地存档；`导出/导入 JSON` 在不同机器间搬电路；`清空` 清空画布。
- **🔗 分享**：把当前电路编码进链接并复制到剪贴板，发给别人直接打开（打开带 `#c=...` 的链接会自动加载）。
- **撤销/重做**：`Ctrl+Z` / `Ctrl+Y`，放心试错。
