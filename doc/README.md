# ChipSim 使用手册

ChipSim 是一个 **2D 逻辑电路沙盒**:在网格上摆元件、连线、跑模拟，搭出从一个与门到加法器、寄存器、乃至更大的数字电路。设计理念是**所见即所得 / 玻璃盒**——结构永远摆在明面上，不藏进黑盒，方便教学和理解。

> 在线地址：把电路用「🔗 分享」生成链接发给别人即可打开。本地改动会自动存到浏览器。

## 一分钟速览

- **底部工具条**选元件，**顶栏**管文件/模板/主题，**右下角**调模拟速度。
- 鼠标左键放置、拖动连续画线；**V** 是选择箭头（默认），**H** 是"操作/探针"模式。
- 想让电路动起来：放 **按钮(B)** 当输入、**灯(L)** 当输出，连上 **门**，切到 **操作模式(H)** 点按钮。
- 不知道某根线现在是几？切 **H**，把鼠标停上去——会点亮整张网并显示值。
