# 5. 搭建复杂电路

复杂电路 = **小积木 + 复用 + 管理布线**。ChipSim 的核心工作流就是：**搭一个小块 → 验证 → 存成模板 → 用模板拼更大的**。

## 一、选区编辑（搬、转、复制、橡皮章）

用 **V（选择）** 工具：

- **点选**单个元件；**Shift 点**加选/减选；空白处**拖动框选**一片；选中后**拖动**整体移动。
- **复制/剪切/粘贴**：`Ctrl+C` / `Ctrl+X` / `Ctrl+V`。粘贴进入"橡皮章"模式——鼠标跟随、**连续点击可盖多份**，`R` 旋转，`Esc` 退出。
- **复制副本** `Ctrl+D`：就地复制一份。
- **整体旋转** `R`；**删除** `Del`。
- 右键任意元件 → 上下文菜单（同样的复制/剪切/副本/粘贴/旋转/删除 + 真值表 + 存为模板）。

这套让你像排版一样摆电路：搭好一组，框选，到处盖。

## 二、模板：把结构变成可复用积木

- **用内置模板**：顶栏 `模板▾` 选 NAND / NOR / XNOR / 半加器 / T 触发器 → 鼠标跟随预览 → 点击盖章（`R` 旋转）。
- **存自定义模板**：用 **V** 框选一块电路 → **右键 → 存为模板** → 起个名。它会出现在 `模板▾` 里（带 ★），可反复盖；不需要时在顶栏点 🗑 删除。
- 模板盖下来是**真实可见、可改**的电路（不是黑盒）——这既是复用，也是学习手段：**盖一个内置模板下来，拆开看它怎么连**。

> 自定义模板和电路一起存在浏览器；导出 JSON 可在不同机器间搬。

## 三、组合逻辑：从门到加法器

回顾几条硬规则（细节见 [工作原理](02-工作原理.md)、[连线·颜色·总线](04-连线-颜色-总线.md)）：

- 与/或/异或是 **2 格高**，左上/左下输入、右输出；两输入**用不同颜色**。
- 一个信号要喂给多个门时，用**同色线**把它拉成一条"母线"再分叉；和别的信号交叉处用**桥**或**异色平行**。

**半加器**（最基础的算术块）：`Sum = A XOR B`，`Carry = A AND B`。

- 直接盖内置 **半加器** 模板最省事；想自己搭就放一个异或 + 一个与，把 A（红）分别喂到两者的一个输入、B（蓝）喂到另一个输入，异或输出 = Sum、与输出 = Carry。
- A、B 要扇出到两个门，会出现"红线和蓝线交叉"——交叉点放一个**桥**。

**全加器 / 多位加法器**：用**组合**而不是从零摆。

- 全加器 = 两个半加器 + 一个或门：
  1. 盖一个半加器，得到 `S1 = A⊕B`、`C1 = A·B`；
  2. 再盖一个半加器，输入 `S1` 和 `Cin`，得到 `Sum = S1⊕Cin`、`C2 = S1·Cin`；
  3. 一个或门：`Cout = C1 + C2`。
  4. 框选整个，**存成"全加器"模板**。
- 4 位行波加法器 = 把"全加器"模板**盖 4 份**，前一位的 `Cout` 接下一位的 `Cin`。多位数据用**总线**搬运、`拆分`/`合并` 在边界转换，能让连线干净很多。

## 四、时序逻辑：记忆、计数、寄存

- **1 位寄存器** = 1 个 **D 触发器(D)**：上=数据 D、下=时钟 CLK、右=Q。时钟上升沿把 D 锁进 Q。
- **n 位寄存器** = n 个 DFF 共用同一根 CLK。
- **T 触发器 / 二分频** = 把 `非(Q)` 接回 D（内置有 **T 触发器** 模板，可直接盖）。每个时钟沿翻转。
- **计数器** = 串联多个 T 触发器，前一级的 Q 当后一级的 CLK（行波计数器），输出就是二进制计数。
- 给时序电路一个 **时钟(K)** 作节拍。**时钟半周期要够长**，让组合逻辑在两次采样之间稳定下来（不对就把时钟调慢，见 [工作原理](02-工作原理.md)）。

## 五、管理复杂度的实用建议

- **自底向上**：先把小块（全加器、寄存器、计数器）搭对、**存成模板**，再用模板拼系统。永远不要一次性摆几百个门。
- **用颜色规划信号**：固定约定（如 A 红、B 蓝、时钟黄、进位绿），一眼看清谁是谁。
- **留间距**：区块之间留空，方便后续框选、移动、加桥。
- **边走边验证**：每搭好一个组合块，框选它 **右键 → 真值表** 核对；时序块用**时序波形**看时钟/触发器对不对（见 [调试与验证](06-调试与验证.md)）。
- **多位用总线**：超过两三位的数据尽量走总线，减少面条。
- **善用撤销**：`Ctrl+Z/Y` 随便试错。
