# ChipSim

2D 逻辑电路沙盒。**混合模型**：表面是格子摆放（Terraria 手感），底层把电路编译成扁平网表，用**同步 delta-step** 内核求值——和 *Turing Complete* 同款模型，反馈回路（锁存器/触发器）天然成立。

目标是沿着 `门 → 加法器 → 寄存器 → RAM → ALU → CPU` 一路爬，最终在格子里造出一台图灵完备的计算机。

## 运行

```bash
npm install
npm run dev        # 开发服务器，监听 0.0.0.0:4000
# 或
npm run build && npm run preview   # 构建后预览，同样 0.0.0.0:4000
```

打开 http://localhost:4000

## 操作（Figma 风）

底部居中浮动工具条（图标 + 悬停提示）；顶部 Nav 卡片（ChipSim 品牌 + 模板 / 撤销重做 / 分享 / 文件 / ?）；右下角速度；选中后右键菜单或快捷键。

| 操作 | 说明 |
| --- | --- |
| `V` 选择 | 点选元件、空白处拖动框选、拖动选中物移动；Shift 加选 |
| 右键 | 弹出菜单：复制/剪切/复制副本/粘贴/旋转/删除 |
| `Ctrl+C/X/V/D` | 复制 / 剪切 / 粘贴 / 复制副本 |
| `R` · `Del` | 旋转选区 · 删除选区 |
| 左栏工具 | `W`线 `A`与 `O`或 `X`异或 `N`非 `B`钮 `L`灯 `K`钟 `D`触发器 `G`桥 `U`总线 `M`合并 `S`拆分 `Y`显示 `E`删除 |

## 架构

```
src/
├─ sim/         纯逻辑，不碰渲染
│  ├─ types.ts      数据模型（Component / Compiled / Op）
│  ├─ geometry.ts   朝向、引脚（terminal）几何
│  ├─ compile.ts    格子电路 → 网表（并查集求连通网）
│  └─ kernel.ts     同步 delta-step 求值器（typed array）
├─ render/      Canvas2D 渲染 + 相机（缩放/平移/裁剪）
├─ editor/      输入/工具/相机交互
├─ ui/          DOM 工具栏
├─ world.ts     格子模型 + 序列化
└─ main.ts      装配 + 主循环（改动即重编译）
```

**关键设计**：一整条连通的线 = 一个网（无长度延迟），每个门 = 1 tick 延迟。每帧跑多 tick（速度可调）。渲染层是独立模块，规模变大后可换 PixiJS 而不动其它部分。

## 设计理念（玻璃盒）

教学 + 娱乐优先，**所见即所得**：结构永远可见，不藏在黑盒里。复用靠**透明模板**（盖章出可见可改的真实电路），不靠不透明封装。布线密度用**彩色线 + 看得见的桥**解决，**不引入分层/3D**。
